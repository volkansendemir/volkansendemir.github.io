Problem #3 of Project Euler series

Question

Project Euler

The prime factors of 13195 are 5, 7, 13 and 29.

What is the largest prime factor of the number 600851475143 ?

Brute-force solution


def brute():
    root = int(num ** (1/2))
    divisors = []
    for i in list(range(2, root))[::-1]:
        if num % i == 0:
            divisors.append(i)
    for divisor in divisors:
        isprime = True
        for ndivisor in divisors:
            if not ndivisor == divisor:
                if divisor % ndivisor == 0:
                    isprime = False
                    break
        if isprime:
            print(divisor)
            return

Although I called this function "brute", it does approach strategically.
Some problems are just too resource-consuming to be solved with crude brute-force.
This is why I have limited my loop to the root of the defined num.
When we look at the divisors of a number (and divisions there of), the divisors start to repeat once we reach its root.
Through the loop, we store each divisor into the divisors list.
Finally we check each divisor against the other divisors to see if they are an exact divisor of each other.
If they are exact divisors, they cannot be prime, so we break out of the loop to save time.
Because we used [::-1] during our loop and we started to loop from the large descending down, we know we have the largest number.
