Distinct primes factors

The first two consecutive numbers to have two distinct prime factors are:

14 = 2 × 7

15 = 3 × 5

The first three consecutive numbers to have three distinct prime factors are:

644 = 22 × 7 × 23

645 = 3 × 5 × 43

646 = 2 × 17 × 19.

Find the first four consecutive integers to have four distinct prime factors each. What is the first of these numbers?

Brute-force solution


def brute():
    i = 0
    while True:
        i = i + 4
        if len(divs(i)) == 4:
            divisors = [0, 0, 0, 0]
            hfd = False
            for j in range(-3, 4):
                divisors = divisors[1:]
                divisors.append(len(divs(i + j)))
                if all(divisor == 4 for divisor in divisors):
                    hfd = True
                    break
            if hfd:
                print(i + j - 4)
                break



def divs(num):
    divisors = []
    root_num = int(num**(1 / 2))
    while num > 1:
        while num % 2 == 0:
            num = int(num / 2)
            if 2 not in divisors: divisors.append(2)
        if len(divisors) > 4: return divisors
        for i in range(3, root_num, 2):
            if num % i == 0:
                divisors.append(i)
                while num % i == 0:
                    num = int(num / i)
        if num >= root_num:
            divisors.append(num)
            return divisors
    return divisors
