Problem #7 of Project Euler series

Question

Project Euler

By listing the first six prime numbers: 2, 3, 5, 7, 11, and 13, we can see that the 6th prime is 13.

What is the 10 001st prime number?

Strategic solution


def strategic():
    primes = [2]
    integer_check = 3
    while len(primes) < lim_up:
        isPrime = True
        for prime in primes:
            if prime > integer_check**(1 / 2): break
            if integer_check % prime == 0: isPrime = False
        if isPrime: primes.append(integer_check)
        integer_check = integer_check + 2
    print(primes[-1])

The only even prime number is 2. Using this knowledge, we can decrease our loop in half by looping through the odd numbers only.
We can go further where we say we need to check only up to the root of each number to ascertain whether its prime or not.
Take 21 for example. Its root is 4.58... so we would only need to check up to 4.
21s divisors up to 4 are only 1 and 3. Using these, we can find that its other divisors are 7 and 21.
We have found all its divisors by only going up to its root value.
Take 17 for example. Its root is 4.123... we would again only need to check up to 4.
17s divisors up to 4 are only 1. We can know for certain that no other divisors exist past the root.

Brute-force solution


def brute():
    primes = [2]
    integer_check = 3
    while len(primes) < lim_up:
        isPrime = all([integer_check % prime != 0 for prime in primes])
        if isPrime: primes.append(integer_check)
        integer_check = integer_check + 2
    print(primes[-1])

In this brute-force solution, I still included the odd-even trick from the strategic approach.
The only difference is that we don't break at the root and instead count up all the way to the value.
Although the odd-even trick cuts the loop in half, it is still very inefficient compared to the stsrategic approach.

Execution

The execution takes multiple seconds, but notice how faster it becomes with the strategic approach.
In fact, the strategic approach can go up to the 80,001st prime by the time it takes the brute-force approach to get to 10,001st.
