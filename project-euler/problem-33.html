Digit cancelling fractions

Solved on Sun, 20 Dec 2020
Uploaded on Sun, 20 Dec 2020

The fraction 49/98 is a curious fraction, as an inexperienced mathematician in attempting to simplify it may incorrectly believe that 49/98 = 4/8, which is correct, is obtained by cancelling the 9s.

We shall consider fractions like, 30/50 = 3/5, to be trivial examples.

There are exactly four non-trivial examples of this type of fraction, less than one in value, and containing two digits in the numerator and denominator.

If the product of these four fractions is given in its lowest common terms, find the value of the denominator.

Brute-force solution


def brute():
    matches = []
    for i in range(lim_bottom, lim_up):
        for j in range(lim_bottom, lim_up):
            if i == j or i > j: continue
            fr = i / j
            it = str(i)[0]
            io = str(i)[1]
            jt = str(j)[0]
            jo = str(j)[1]
            if it == jt and it != "0" and jo != "0":
                if (int(io) / int(jo)) == fr:
                    matches.append([int(io), int(jo)])
            elif io == jt and io != "0" and jo != "0":
                if (int(it) / int(jo)) == fr:
                    matches.append([int(it), int(jo)])
            elif io == jo and io != "0" and jt != "0":
                if (int(it) / int(jt)) == fr:
                    matches.append([int(it), int(jt)])
            elif jo == it and it != "0" and jt != "0":
                if (int(io) / int(jt)) == fr:
                    matches.append([int(io), int(jt)])
    print(matches)
