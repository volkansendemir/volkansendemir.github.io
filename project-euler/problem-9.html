A Pythagorean triplet is a set of three natural numbers, a < b < c, for which, a2 + b2 = c2

For example, 32 + 42 = 9 + 16 = 25 = 52.

There exists exactly one Pythagorean triplet for which a + b + c = 1000.

Find the product abc.

Strategic solution


def strategic():
    for i in range(1, (lim_up + 1)):
        for j in range(i + 1, (lim_up - 2 - (i * 3))):
            k = lim_up - i - j
            if (i**2) + (j**2) == k**2:
                print(i, j, k)
                print(i*j*k)
                return

Brute-force solution


def brute():
    for i in range(1, (lim_up + 1)):
        for j in range(i, (lim_up + 1)):
            for k in range(j, (lim_up + 1)):
                if i + j + k == lim_up:
                    if (i**2) + (j**2) == k**2:
                        print(i, j, k)
                        print(i*j*k)
                        return
