Sub-string divisibility

Solved on Mon, 21 Dec 2020
Uploaded on Tue, 22 Dec 2020

The number, 1406357289, is a 0 to 9 pandigital number because it is made up of each of the digits 0 to 9 in some order, but it also has a rather interesting sub-string divisibility property.

Let d1 be the 1st digit, d2 be the 2nd digit, and so on. In this way, we note the following:

  • d2d3d4=406 is divisible by 2
  • d3d4d5=063 is divisible by 3
  • d4d5d6=635 is divisible by 5
  • d5d6d7=357 is divisible by 7
  • d6d7d8=572 is divisible by 11
  • d7d8d9=728 is divisible by 13
  • d8d9d10=289 is divisible by 17

Find the sum of all 0 to 9 pandigital numbers with this property.

Brute-force solution


def brute():
    result = 0
    primes = [2, 3, 5, 7, 11, 13, 17]
    divisors = [[] for prime in primes]
    versions = []
    combs = combinations(10, 3)
    for comb in combs:
        perms = permutation(comb)
        for perm in perms:
            perm_str = "".join([str(p - 1) for p in perm])
            if perm_str not in versions:
                versions.append(perm_str)
    for prime in primes:
        for version in versions:
            if int(version) % prime == 0:
                divisors[primes.index(prime)].append(version)

    digits = [str(digit) for digit in range(10)]
    for a in divisors[0]:
        for b in divisors[1]:
            if a[1:] == b[:-1]:
                for c in divisors[2]:
                    if b[1:] == c[:-1]:
                        for d in divisors[3]:
                            if c[1:] == d[:-1]:
                                for e in divisors[4]:
                                    if d[1:] == e[:-1]:
                                        for f in divisors[5]:
                                            if e[1:] == f[:-1]:
                                                for g in divisors[6]:
                                                    if f[1:] == g[:-1]:
                                                        reps = 0
                                                        missing = 0
                                                        for digit in digits:
                                                            if digit in f"{a}{d}{g}": reps = reps + 1
                                                            else: missing = missing + int(digit)
                                                        if reps == 9:
                                                            print(f"{missing}{a}{d}{g}")
                                                            result = result + int(f"{missing}{a}{d}{g}")
    print(result)



def permutation(s):
    if len(s) == 1:
        return [s]
    perm_list = []
    for a in s:
        remaining_elements = [x for x in s if x != a]
        z = permutation(remaining_elements)
        for t in z:
            perm_list.append([a] + t)
    return perm_list


def combinations(n, k, min_n=0, accumulator=None):
    if accumulator is None:
        accumulator = []
    if k == 0:
        return [accumulator]
    else:
        return [l for x in range(min_n, n) for l in combinations(n, k - 1, x + 1, accumulator + [x + 1])]
