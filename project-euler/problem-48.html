The series, 11 + 22 + 33 + ... + 1010 = 10405071317.

Find the last ten digits of the series, 11 + 22 + 33 + ... + 10001000.

Brute-force solution


def brute():
    s = 0
    for i in range(1, lim_up + 1):
        s = s + i**i
    print(str(s)[:-11:-1][::-1])

Execution


lim_up = 1000

brute()

#111657th

Output


brute: 9110846700
