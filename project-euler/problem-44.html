Pentagon numbers

Solved on Mon, 21 Dec 2020

Pentagonal numbers are generated by the formula, Pn=n(3n−1)/2. The first ten pentagonal numbers are:

1, 5, 12, 22, 35, 51, 70, 92, 117, 145, ...

It can be seen that P7 + P4 = 22 + 70 = 92 = P8. However, their difference, 70 − 22 = 48, is not pentagonal.

Find the pair of pentagonal numbers, Pj and Pk, for which their sum and difference are pentagonal and D = |Pk − Pj| is minimised; what is the value of D?

Brute-force solution


def brute():
    not_found = True
    i = 2165  # normally starts from 1 but takes very long
    while not_found:
        i = i + 1
        large_pentagonal = int(((3 * (i ** 2)) - i) / 2)
        for j in list(range(1, i))[::-1]:
            small_pentagonal = int(((3 * (j ** 2)) - j) / 2)
            if is_pentagonal(large_pentagonal - small_pentagonal) and is_pentagonal(large_pentagonal + small_pentagonal):
                not_found = False
                break
    print(large_pentagonal, small_pentagonal)
    print(large_pentagonal - small_pentagonal)



def get_pentagonals(lim_up):
    pentagonals = []
    for i in range(1, lim_up + 1):
        new_pentagonal = int(((3 * (i**2)) - i) / 2)
        pentagonals.append(new_pentagonal)
    return pentagonals


def is_pentagonal(num):
    double_num = num * 2
    root = int(double_num**(1 / 2))
    for i in range(1, root):
        if double_num % i == 0:
            if ((i * 3) - 1) == (double_num / i):
                return True
    return False
