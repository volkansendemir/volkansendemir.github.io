Let d(n) be defined as the sum of proper divisors of n (numbers less than n which divide evenly into n).

If d(a) = b and d(b) = a, where a â‰  b, then a and b are an amicable pair and each of a and b are called amicable numbers.

For example, the proper divisors of 220 are 1, 2, 4, 5, 10, 11, 20, 22, 44, 55 and 110; therefore d(220) = 284. The proper divisors of 284 are 1, 2, 4, 71 and 142; so d(284) = 220.

Evaluate the sum of all the amicable numbers under 10000.

Brute-force solution


def brute():
    computed = []
    for i in range(lim_up):
        computed.append(False)
    amic_sum = 0
    for i in range(1, lim_up):
        if not computed[i]:
            sum_divisors = sum(get_divisors(i))
            computed[i - 1] = True
            if sum_divisors < lim_up:
                if not computed[sum_divisors - 1]:
                    sum_target = sum(get_divisors(sum_divisors))
                    if i == sum_target:
                        amic_sum = amic_sum + i + sum_divisors
                        computed[sum_divisors - 1] = True
    print(amic_sum)



def get_divisors(number):
    divisors = [1]
    for i in range(2, int(number**(1 / 2)) + 1):
        if number % i == 0:
            divisors.append(i)
            if i != int(number / i):
                divisors.append(int(number / i))
    return divisors
