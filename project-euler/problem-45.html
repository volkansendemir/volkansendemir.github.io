Triangular, pentagonal, and hexagonal

Solved on Mon, 21 Dec 2020
Uploaded on Tue, 22 Dec 2020

Triangle, pentagonal, and hexagonal numbers are generated by the following formulae:

Triangle	 	Tn=n(n+1)/2		1, 3, 6, 10, 15, ...
Pentagonal 	Pn=n(3n−1)/2	1, 5, 12, 22, 35, ...
Hexagonal 	Hn=n(2n−1)		1, 6, 15, 28, 45, ...

It can be verified that T285 = P165 = H143 = 40755.

Find the next triangle number that is also pentagonal and hexagonal.

Brute-force solution


def brute():
    for i in range(40756, 100000):
        triangular = int((i * (i + 1)) / 2)
        if is_pentagonal(triangular) and is_hexagonal(triangular): break
    print(triangular)



def is_pentagonal(num):
    double_num = num * 2
    root = int(double_num**(1 / 2))
    for i in range(1, root):
        if double_num % i == 0:
            if ((i * 3) - 1) == (double_num / i):
                return True
    return False


def is_hexagonal(num):
    root = int(num**(1 / 2))
    for i in range(1, root):
        if num % i == 0:
            if ((2 * i) - 1) == (num / i):
                return True
    return False
