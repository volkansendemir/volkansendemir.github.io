Goldbach's other conjecture

Solved on Tue, 22 Dec 2020
Uploaded on Tue, 22 Dec 2020

It was proposed by Christian Goldbach that every odd composite number can be written as the sum of a prime and twice a square.

  • 9 = 7 + 2×12
  • 15 = 7 + 2×22
  • 21 = 3 + 2×32
  • 25 = 7 + 2×32
  • 27 = 19 + 2×22
  • 33 = 31 + 2×12

It turns out that the conjecture was false.

What is the smallest odd composite that cannot be written as the sum of a prime and twice a square?

Brute-force solution


def brute():
    primes = [2]
    j = 100
    while j < 100000:
        for i in range(j - 97, j, 2):
            if is_prime(i, primes): primes.append(i)
        for i in range(j - 97, j, 2):
            if i not in primes:
                if not is_goldbach(i, primes):
                    print(i)
        j = j + 100



def is_prime(num, primes):
    for prime in primes:
        if num % prime == 0: return False
        if prime > num: return True
    return True


def is_goldbach(num, primes):
    a = int((num / 2)**(1 / 2)) + 1
    for i in range(1, a):
        if (num - ((i**2) * 2)) in primes: return True
    return False
